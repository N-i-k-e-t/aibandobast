# AI BANDOBaST

This repository contains the **AI BANDOBaST Internal Administrative Portal**.

## Quick Start (Vercel Showcase)
1. Copy data to `portal/data/inbox/`.
2. See [portal/README_SHOWCASE.md](./portal/README_SHOWCASE.md) for full setup.

## Project Structure
- `portal/`: Next.js application (The main portal).
- `portal/data/inbox/`: Place raw files here for auto-indexing.
- `portal/scripts/`: Build-time manifest generator.
